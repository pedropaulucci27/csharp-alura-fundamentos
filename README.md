# C# — Criando sua Primeira Aplicação

Repositório com exercícios e projetos desenvolvidos durante o curso **C#: Creating Your First Application** da [Alura](https://www.alura.com.br), ministrado pelos instrutores [Guilherme Lima](https://linktr.ee/guilimadev) e [Daniel Portugal](https://cursos.alura.com.br/user/daniel-portugal).

---

## 📚 Sobre o curso

Curso de 8 horas focado nos fundamentos de C# e .NET, cobrindo desde os primeiros passos com a linguagem até estruturas de dados e um desafio prático final.

**Módulos:**
- Criando o primeiro programa
- Tipos e funções no C#
- Listas e loops no C#
- Dicionários e avaliação de bandas
- Desafio final

## 🧠 Conceitos praticados

- Tipos de dados (`int`, `float`, `string`, `bool`)
- Estruturas de controle (`if/else`, `switch`, `for`, `foreach`, `do/while`)
- Funções e métodos com parâmetros
- Interpolação de strings (`$"..."`)
- Coleções: `List<T>`, `Dictionary<K,V>`, `HashSet<T>`
- Manipulação de listas e dicionários
- Entrada e saída de dados via console
